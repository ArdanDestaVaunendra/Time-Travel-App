# Magic Fake Lock Screen

A specialized Android application developed for magic routines and illusions. This app mimics a realistic Samsung One UI lock screen, allowing for seamless integration into sleight-of-hand performances.

## Features
- **Authentic UI:** Pixel-perfect Samsung lock screen replication using `SamsungOne` fonts.
- **Dynamic Battery Status:** A custom-calibrated battery gauge that can be manipulated for "force" effects.
- **Hidden Triggers:** Support for secret gestures to reveal specific information (like the Queen of Hearts).
- **Custom Marquee:** Dynamic bottom text for personalized reveals.

## Tech Stack
- **Language:** Kotlin
- **Layout:** Android XML (ConstraintLayout, Custom Drawables)
- **UI/UX:** Inspired by Samsung One UI
